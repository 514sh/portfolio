# MARK BALAGTAS - PORTFOLIO

## Prerequisites

- node >= 22.x.x
- miktex >= 25.x.x

## Installing the dependencies

```bash
npm i
# or
pnpm i
```

## Getting Started

Running the development server:

```bash
npm run dev
# or
pnpm dev
```

Generating the cv pdf file:

```bash
npm run gen-pdf
# or
pnpm gen-pdf
```
